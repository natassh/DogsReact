Proyecto desarrollado con:

  • ES6

    En este proyecto uso Arrow Functions, const, let, Async/Await, ES6 Modules, Fetch, etc.

  • Webpack

    Webpack como module bundle aprendiendo a configurarlo a mano.

  • Tooling

    ESLint y Netlify para el deploy.

  • CSS

    Usando metodología BEM y Atomic Design para la división de los estilos. Con PostCSS y Custom properties (variables CSS) para escribir CSS moderno.

  • Html5

    Marcado semántico (aside, section, figure, etc.)
